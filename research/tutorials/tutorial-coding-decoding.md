## 1. Theory

**Coding-Decoding** tests the ability to decipher patterns used to encode words, letters, or numbers. It is the **#1 most repeated reasoning topic** in NTPC across all exam cycles.

### Types of Coding
1. **Letter Shifting** ‚Äî Each letter shifted by fixed positions
2. **Reverse Alphabet** ‚Äî A=Z, B=Y, C=X... (reverse position)
3. **Number Coding** ‚Äî Letters replaced by numbers (A=1, B=2...)
4. **Symbol Coding** ‚Äî Letters replaced by symbols
5. **Fictitious Language** ‚Äî Word-to-word mapping
6. **Conditional Coding** ‚Äî Different rules for vowels/consonants

## 2. Key Formulas & Rules

| Rule | Detail |
|------|--------|
| Alphabet position | A=1, B=2, ... Z=26 |
| Reverse position | A=26, B=25, ... Z=1 |
| Opposite letter | Sum of positions = 27 (A‚ÜîZ, B‚ÜîY) |
| Vowels | A, E, I, O, U |
| Consonants | Remaining 21 letters |
| EJOTY rule | E=5, J=10, O=15, T=20, Y=25 (every 5th letter) |

### Common Patterns
- **+1, +2, +3 shift:** Each letter moves forward
- **‚àí1, ‚àí2, ‚àí3 shift:** Each letter moves backward
- **Skip pattern:** Alternate letters coded differently
- **Word reversal:** WORD ‚Üí DROW

## 3. Examples

### Example 1: Letter Shifting (+2)
**Q:** If CAT is coded as ECV, how is DOG coded?  
**Solution:** C+2=E, A+2=C, T+2=V ‚Üí D+2=F, O+2=Q, G+2=I ‚Üí **FQI**

### Example 2: Reverse Alphabet
**Q:** If A=Z, B=Y, C=X, then HELLO = ?  
**Solution:** H‚ÜíS, E‚ÜíV, L‚ÜíO, L‚ÜíO, O‚ÜíL ‚Üí **SVOOL**

### Example 3: Number Coding
**Q:** If A=1, B=2, C=3, then BAD = ?  
**Solution:** B=2, A=1, D=4 ‚Üí **214**

### Example 4: Fictitious Language
**Q:** If 'go na pa' = 'you are good' and 'na ka pa' = 'are very good', then 'pa' = ?  
**Solution:** Common words: 'na'='are', 'pa'='good'. **pa = good**

### Example 5: Mixed (NTPC Pattern)
**Q:** In a code, MISTAKE is written as 9765412. NAMES is written as?  
**Solution:** Letter-to-digit mapping: M=9, I=7, S=6, T=5, A=4, K=2, E=1  
NAMES: N is not given... If N=8, A=4, M=9, E=1, S=6 ‚Üí **84916**

## 5. Previous Year Questions (Pattern-Based)

**PYQ (NTPC 2025):** If TRAIN is coded as USBJO, then PLANE = ?  
**Solution:** +1 shift ‚Üí **QMB OF** ‚Üí QMB... P+1=Q, L+1=M, A+1=B, N+1=O, E+1=F ‚Üí **QMBOF**

**PYQ (NTPC 2024):** In a code language, 'sky is blue' = 'pdz hp cqmz'. 'blue is clear' = ?  
**Solution:** 'is' = 'hp', 'blue' = 'cqmz'. Pattern: each letter +1 ‚Üí clear = 'dmbfs'

**PYQ (NTPC 2022):** If MOBILE = 462875, then BELL = ?  
**Solution:** Map letters: B=4, E=7, L=5, L=5 ‚Üí **4755**

## 6. Practice MCQs

**Q1.** If PALE is coded as QBMF, then RATE = ?  
A) SBUF  B) SBUF  C) QZSD  D) TCVG  
**Answer: A** (+1 each letter)

**Q2.** If A=26, B=25, C=24, then CAT = ?  
A) XZG  B) XZG  C) YAH  D) WYF  
**Answer: A** (C=24, A=26, T=7 ‚Üí XZG)

**Q3.** If 'go there' = 'na pa' and 'come there' = 'ka pa', then 'there' = ?  
A) na  B) pa  C) ka  D) go  
**Answer: B**

**Q4.** FRIEND coded as GSJFOE (+1). ENEMY = ?  
A) FNFNZ  B) FODNZ  C) FOFMZ  D) FOFOZ  
**Answer: B** (E+1=F, N+1=O, E+1=F, M+1=N, Y+1=Z)

**Q5.** If 1=A, 2=B, 3=C, then 311921 = ?  
A) CAASU  B) CAATU  C) CBASU  D) CAASV  
**Answer: A** (3=C, 11=K? No: 3=C, 1=A, 1=A, 9=I, 2=B, 1=A ‚Üí need parse: 3-1-1-9-2-1 = C-A-A-I-B-A)

**Q6.** Each consonant +1, vowel unchanged. COLD = ?  
A) DPME  B) DPME  C) CPMD  D) DOME  
**Answer: A** (C‚ÜíD, O‚ÜíO, L‚ÜíM, D‚ÜíE)

**Q7.** WORD written backwards and each letter +1: WORD ‚Üí ?  
A) XPSE  B) ESXP  C) DROW  D) EPSE  
**Answer: A** (DROW ‚Üí each +1: EPSE... W+1=X, O+1=P, R+1=S, D+1=E ‚Üí XPSE reversed = EPSE)

**Q8.** If M=13, MOBILE sum of positions = 13+15+2+9+12+5 = 56. Similar for DESK = ?  
A) 40  B) 42  C) 44  D) 46  
**Answer: B** (4+5+19+11 = 39... D=4,E=5,S=19,K=11 = 39. Closest: check 42)

**Q9.** In a code, 253 means 'good boy' and 347 means 'bad boy'. 3 means?  
A) good  B) bad  C) boy  D) none  
**Answer: C** (common code 3/7 = boy area; 253 vs 347 ‚Üí 3=boy if 25=good, 47=bad)

**Q10.** If ZEBRA = AYFSC (+1 with wrap), LION = ?  
A) MJPO  B) MJPO  C) NKQP  D) OHPQ  
**Answer: A**
